📌 Overview
Slide Library — это корпоративная библиотека слайдов, встроенная прямо в PowerPoint в виде надстройки (Office Add-in).
Решение позволяет пользователям быстро находить, просматривать и вставлять утверждённые корпоративные слайды, снижая время на создание презентаций и повышая консистентность визуальных материалов внутри компании.
🎯 Problem Statement
Внутри компаний создание презентаций часто сопровождается следующими проблемами:
отсутствует единый источник “approved slides”
сотрудники пересобирают одинаковые слайды вручную
теряется консистентность бренда и структуры презентаций
большое количество времени тратится на поиск и копирование материалов
💡 Solution
Мы разработали встроенную библиотеку слайдов внутри PowerPoint, которая решает эти проблемы.
Основные возможности:
📚 централизованный каталог слайдов
🔍 поиск по названию и тегам
🏷 фильтрация по категориям (Business, Analytics, Marketing, Product)
⭐ featured-секция (рекомендованные слайды)
🔥 аналитика использования (mock data)
➕ вставка слайда в презентацию в 1 клик
🧭 User Flow
Пользователь открывает PowerPoint
Запускает надстройку Slide Library
Просматривает каталог слайдов
Использует поиск или фильтры
Открывает превью слайда
Нажимает Insert slide
Слайд добавляется в презентацию
